Techno Games

Treasure hunt using TAGHUNT. A technology based treasure hunt.

Batoota owns TAGHUNT where you can hunt for tags that are strategically hidden at various locations. We provide you with a device that can detect and read the Tag when it is in the vicinity. The winning team must detect all the tags in the proper order.

Batoota games also has Interactive floors, RFID based games, Gesture based games that use Kinect and play station, leap motion devices, Sphero robotic balls .
